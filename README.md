# Finance App - Spending & Income Analysis

Analyze spending and income across multiple bank accounts with different currencies.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

### Version 1: Load & Convert Data
```bash
python main_v1.py
```

### Version 2: Category Analysis
```bash
python main_v2.py
```

### Version 3: Full Analytics + Visualization
```bash
python main_v3.py
```
